# 1. 初始化ts项目
1. npm init : 创建package.json
2. tsc --init : 创建 tsc config.json
3. tsc 文件名.ts 编译出js 文件

# webpack 配置
> ts-loader
```
module: {
    rules: [
        {
            test: /\.tsx?$/i,
            use: [{
                loader: 'ts-loader'
            }],
            exclude: /node_modules/
        }
    ]
},
```
